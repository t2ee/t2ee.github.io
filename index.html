t2ee - Typescript To Enterprise Edition

  • Core

    @t2ee/core implements the most crucial functionalities for all the rest libraries, including dependency injection, auto configuration and bean management.

    GitHub

  • Sl4js

    @t2ee/sl4js is a logging library supports custom appenders, rich pattern (logging format), configurable loggers, etc.

    GitHub

  • Vader

    @t2ee/vader is a router component suitable for Koa 2. This library makes full usage of decorators to empower development fluency.

    GitHub

  • Deathstar

    @t2ee/deathstar is a bootstrap helper, it automatically scans & load all your files in your current folder and subfolders, with the help of @t2ee/core, components can be automatically injected without any configurations.

    GitHub

  • Validation

    @t2ee/validation is a simple yet strong form validation library.

    GitHub

  • Restful

    @t2ee/restful is a restful client, makes it a lot easier to request existing resources and third-party apis.

    GitHub
